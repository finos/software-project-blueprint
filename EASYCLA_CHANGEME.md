This file is used to test EasyCLA onboarding.

To get started, [just submit a Pull Request](https://github.com/finos/software-project-blueprint/edit/main/EASYCLA_CHANGEME.md) with a simple change to this file (adding an empty line, or some random text at the bottom); this will trigger the EasyCLA bot, which will post a comment on the pull request stating whether all contributors are covered by the project’s **CLA** requirements; if not, the bot will post instructions on how to sign.

You can read more about EasyCLA on the [FINOS Community site](https://community.finos.org/docs/governance/Software-Projects/easycla). For how CLA relates to **DCO** and Labs defaults, see [Contribution Requirements](https://community.finos.org/docs/governance/Software-Projects/contribution-compliance-requirements).
